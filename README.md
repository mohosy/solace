# Solace

A queue-processing simulator for modeling distributed consumers, retries, dead-letter behavior, and throughput/latency tradeoffs.

## Highlights
- Simulated multi-consumer scheduling and message processing timeline.
- Retry and DLQ behavior under configurable failure rates.
- Auto-tuning mode for adaptive consumer count selection.
- Metrics dashboard with throughput, p95 latency, retries, and DLQ count.

## Stack
- HTML5 Canvas
- Vanilla JavaScript
- CSS dashboard UI

## Run
Open `index.html` in your browser.
